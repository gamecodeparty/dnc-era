# Executa um PLAN.md

- **$ARGUMENTS**: Conjunto de tarefas a executar ou instruções adicionais

## Instruções

1. Leia o arquivo PLAN.md para identificar as tarefas definidas
2. Interprete as instruções fornecidas no arquivo PLAN.md
3. Execute as tarefas interpretadas conforme especificado no arquivo PLAN.md
4. Verifique se todas as tarefas foram concluídas com sucesso
5. Marque as tarefas concluídas no arquivo PLAN.md a medida em que as resolve

## Observações

- **Não gere testes unitários ou testes automatizados**
- **NÃO É NECESSÁRIO produzir sumários ou relatórios de execução da tarefa.**
